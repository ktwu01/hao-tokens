# Code of conduct

We follow the [Contributor Covenant](https://www.contributor-covenant.org/), version **2.1**.  
The full text is available at [contributor-covenant.org/version/2/1/code_of_conduct/](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

## Reporting

Open a GitHub issue (or contact maintainers privately if needed) for conduct concerns related to this project’s issues, discussions, or pull requests.
