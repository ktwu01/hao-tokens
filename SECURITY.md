# Security

This project is a **static documentation and link index**. It does not ship executable services or handle user secrets.

## Reporting a concern

If you believe content in this repository introduces a **security or abuse** risk (for example, links that promote malware or facilitate account theft), please **open a GitHub issue** with:

- The exact file and section (or link) involved
- Why it is harmful
- Any supporting references (optional)

For **vulnerabilities in GitHub itself** or your account, use [GitHub’s own reporting channels](https://github.com/security).

Do **not** submit real API keys, passwords, or personal data in public issues.
