# Contributing

Thanks for helping keep **hao-tokens** accurate and useful.

## What we want

- **Official links only** for student or educator programs (vendor pages, government education portals, or clearly documented free tiers).
- **Dated updates** when an offer ends (e.g. “Offer ended 2026‑03; see archive.org link if you have one”).
- **New GitHub lists** that are actively maintained and clearly describe **provider-published** rate limits or eligibility.

## What we do not want

- Shared **paid API keys**, account dumps, or credential trading.
- Instructions whose **main goal** is breaking a provider’s Terms of Service or bypassing billing.
- Affiliate or **spam** links disguised as “student deals.”

## How to contribute

1. Open an **issue** for discussion if the change is large or contentious.
2. Prefer **small PRs**: one logical change (e.g. “fix Perplexity education URLs” or “add one new awesome list”).
3. For new rows in the README table, include **role** (⭐ / 🎓 / 🛠 / ⚠ / 📚) consistent with the legend.
